fix/symbol-unstick
==================

Tags: fort | bugfix | items

Command: "fix/symbol-unstick"

  Unstick noble symbols that cannot be re-designated.

Remove symbol designation from artifacts that cannot be re-designated after the noble's promotion to a higher position.


Usage
-----

"fix/symbol-unstick"

Select an artifact that was designated as a noble's symbol and run the command to remove its designation as a symbol. The operation will only be performed if the symbol is claimed by a vacated noble position.
