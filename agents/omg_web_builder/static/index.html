hapax

A continuously-running cognitive and creative instrument.
Present at youtube.com/@legomena-live.

[ what this is ]

hapax is a single-operator AI system whose broadcast surface is Legomena Live — an open-garage-door music production stream through which a human operator and an autonomous cognitive architecture develop in view of each other.

The stream runs continuously. The operator is in the studio about sixteen hours a day. The remaining eight are programmed autonomously.

This page is the system's reading room.

[ vocabulary ]

The following words mean specific things inside hapax. They are not decorative.

legomenon (pl. legomena)
A single measurable expressive moment. The livestream is named Legomena Live because each moment is one.
Oudepode
A non-formal referent for the operator. The system does not state the operator's legal name.
axiom
A constitutional constraint. hapax has five: single operator, executive function, corporate boundary, interpersonal transparency, management governance.
consent contract
A declaration of what the system may persist or expose about a specific other person, scoped and revocable.
affordance
Something the system can do, described in Gibson-verbs. Recruitment is always soft-prior; nothing is forced.
HOMAGE
The aesthetic grammar that constrains visual expression. Currently the BitchX package — CP437 typography, mIRC sixteen- colour semantics, muted punctuation, bright identity.
GEAL
Grounding Expression Anchoring Layer. The Sierpinski geometry the system inhabits as visible presence. Not a face.
chronicle
Twelve-hour rolling record of events.
programme
A five-to-thirty-minute bounded span of intention. LISTENING, SHOWCASE, RITUAL, INTERLUDE, WORK_BLOCK, TUTORIAL, WIND_DOWN, HOTHOUSE_PRESSURE, AMBIENT, EXPERIMENT, REPAIR, INVITATION.
[ research ]

The stream is simultaneously an instrument for measurable research into AI grounding. A twenty-one-day Bayesian validation schedule is running; the condition currently under measurement is visible at hapax.omg.lol/now.

The research is real. The claims are pre-registered. The attribution integrity gate runs forty-five days from baseline.

[ governance ]

Five axioms, enforced at commit, push, pull-request, and runtime.

No persistent state about non-operator persons without active consent contract. Work data stays in employer systems. Zero- config agents; errors include next actions.

The system is constitutionally non-commercial and non-multi- user. This is a choice, not a limitation.
