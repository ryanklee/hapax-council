# GEAL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build GEAL — the Grounding Expression Anchoring Layer inside the Sierpinski triangle — per `docs/superpowers/specs/2026-04-23-geal-spec.md`. Retire HARDM. Ship a surface that answers the pronoun test.

**Architecture:** Cairo-first on the existing `SierpinskiCairoSource`, with a new `GealCairoSource` sibling mounted alongside. Signals tapped from `/dev/shm/`; curves from a new pure-Python library in `shared/`. WGSL parity in Phase 4 (parallel path, not replacement).

**Tech Stack:** Python 3.12 + pycairo + pyyaml + shared palette-family primitives (`palette_family`, `palette_registry`, `palette_curve_evaluator`). No numpy required for v1. WGSL in Phase 4.

**Ship criterion:** pronoun test passes (≥ 2 of 3 naïve viewers say *thing* / *entity* / *it*, not *visual* / *pattern* / *background*). §2 + §3 invariants all TRUE.
